
# ILEARN Analytics Tool v2

This Streamlit app allows teachers to upload student report PDFs and generate performance statistics.

## Features
- Upload multiple PDFs
- Parse student names, Lexile scores, and performance levels
- Dashboard cards for class metrics
- Class overview table
- Per-student breakdown
- Download as CSV and PDF
- Custom school name and logo in header and PDF

## Deployment Instructions
1. Create a GitHub repository and upload:
   - app.py
   - requirements.txt
   - README.md
2. Go to https://streamlit.io/cloud and sign in with GitHub.
3. Click **New App** and select your repository.
4. Set **Main file path** to `app.py`.
5. Click **Deploy**.
6. Share the generated URL with teachers.
